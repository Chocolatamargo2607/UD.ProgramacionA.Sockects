El programa consta de dos aplicaciones: el Front-End (cliente) y el Back-End (servidor). Para iniciar el servidor, se debe ejecutar el comando "npm start". El proceso de despliegue del servidor comienza con la detección de los puertos físicos disponibles en el equipo (servidor), utilizando la librería OS de Node.js. Si no se encuentra un puerto Ethernet activo, se desplegará el servidor utilizando la dirección IPV4 del WiFi, la cual se puede encontrar en el archivo "admIP". Una vez que el servidor ha sido desplegado, la consola del servidor mostrará un mensaje que incluye la dirección IP del servidor y el puerto lógico en el que se ha desplegado. A través de este puerto, el cliente puede acceder al servidor.

El archivo "index.js" contiene la aplicación del Back-End (servidor), mientras que el archivo "index.html" y "app.js" contienen la aplicación del Front-End (cliente).
